Class JXG.Curve

JXG.GeometryElement
      ↳ JXG.Curve

Creates a new curve object. Do not use this constructor to create a curve. Use JXG.Board#create with type Curve, or Functiongraph instead.

Defined in: curve.js.
Extends JXG.GeometryElement.
